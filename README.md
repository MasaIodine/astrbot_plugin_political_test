# AstrBot 8values 测试软件

一个用于 AstrBot (v3) 的意识形态测试插件。

## 功能
- 提问并自动测定受试者最接近的意识形态。
- 可以将结果渲染为图片。

- 1.0.1 新功能：测试完成后，调用LLM自动进行评价

## 使用方法
- valuetest 开始测试
- valueans [1-5] 对问题作答
- valuestop 退出测试

## 安装方法
在 AstrBot 管理面板的插件市场中，点击“通过链接安装”，输入此仓库地址：
`https://github.com/MasaIodine/astrbot_plugin_political_test`
